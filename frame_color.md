# 颜色属性：border-color

## 介绍

控制边框颜色，并且有了更大的灵活性，可以产生渐变效果。

## 语法

```
border-color：<color>
```


例：

```javascript
border-color:<color>
/*其中可以上一个值，也可以是多个值*/
/*分别给各边上色*/
border-top-color:<color>/*给上边框上色*/
border-right-color:<color>/*给右边框上色*/
border-bottom-color:<color>/*给下边框上色*/
border-left-color:<color>/*给左框上色*/
```
