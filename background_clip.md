# 裁切属性：background-clip

## 介绍

设置对象的背景向外裁剪的区域

例：

```javascript
<!DOCTYPE html>
<html>
<head>
<style>
div
{
width:300px;
height:100px;
padding:50px;
background-color:yellow;
background-clip:content-box;
border:2px solid #92b901;
}
</style>
</head>
<body>

<div>
极客学院极客学院极客学院极客学院极客学院极客学院极客学院极客学院

</body>
```
